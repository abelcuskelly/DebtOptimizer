DebtOptimizer Icon PNG Generator

To create a PNG version of the icon:

  1. Right-click on the icon below
  2. Select "Save image as..." or take a screenshot
  3. Save as "icon-192.png"
  4. Replace the placeholder file in the public directory

Alternative: Online Conversion

You can also:

  • Visit Convertio SVG to PNG

  • Download the converted PNG
  • Replace the file in your project
